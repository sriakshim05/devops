## Objective

The objective of this task is to provision and manage cloud infrastructure using Terraform as an Infrastructure as Code (IaC) tool. Terraform enables automated, consistent, repeatable, and scalable infrastructure deployments.

---

## About Terraform

Terraform is an open-source Infrastructure as Code (IaC) tool developed by HashiCorp. It allows users to define and manage cloud infrastructure using declarative configuration files.

Key Benefits:

- Automates infrastructure provisioning
- Supports multiple cloud providers (AWS, Azure, GCP)
- Enables version-controlled infrastructure
- Provides consistent and repeatable deployments
- Uses declarative configuration language (HCL)
